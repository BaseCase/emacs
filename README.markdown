cjb's Emacs setup
=================

My personal Emacs configuration.

### Setting up ###

1. `mv .emacs ~`
1. Make sure you have these fonts installed: http://font.ubuntu.com/
(or pick your own)


### Flyspell setup ###

If on a Mac, you need Homebrew.

1. `brew remove aspell`
1. `brew install aspell --lang=en`
